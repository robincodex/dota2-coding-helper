# DOTAItemImage

物品图标，继承至`Image`，只能显示物品图标

## 属性

- XML属性


| 属性名        | 类型      | 描述  |
| ------------- |:--------:| ----- |
| itemname   | String  | 通过`AbilityName`显示技能的图标 |


- Javascript属性


| 属性名               | 类型      | 描述  |
| -------------------- |:--------:| ----- |
| itemname             | String   | 通过`AbilityName`显示技能的图标 |
| contextEntityIndex   | Integer  | 通过`EntityIndex`显示技能的图标 |
