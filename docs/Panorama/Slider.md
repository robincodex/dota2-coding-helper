# Slider

滑动条

## 属性

| 属性名        | 类型      | 描述  |
| ------------- |:--------:| ----- |
| direction     | String  | 方向，`horizontal`表示水平（需要`class="HorizontalSlider"`配合），`vertical`表示垂直（默认） |


## JS 属性

| 属性名        | 类型      | 描述  |
| ------------- |:--------:| ----- |
| min           | Float    | 最小值，默认0.0 |
| max           | Float    | 最大值，默认1.0 |
| value         | Float    | 值     |
| default       | Float    | 默认值 |
| increment     | Float    | 递增值，默认0.10 |
